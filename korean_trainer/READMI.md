# Korean Vocabulary Trainer 🇰🇷 / Тренажёр корейских слов 📖
### 🇬🇧 English

A simple console program to practice Korean–English vocabulary.  
It shows a Korean word, asks for the English translation, checks the answer, and keeps score.

**Features:**
- 50+ basic Korean words with translations
- Tracks correct and wrong answers
- Friendly feedback after each attempt
