# Gratitude Journal / Дневник благодарностей 🌺🎉

A simple console program to record daily gratitudes with automatic date and view past entries.

Простая консольная программа для записи благодарностей с автоматической датой и просмотром прошлых записей.

**Features / Возможности:**
- Add a gratitude with today's date / Добавить благодарность с сегодняшней датой
- Save all entries to a file / Сохранять записи в файл
- View last 5 entries / Просмотреть последние 5 записей

**How to run / Как запустить:**
```bash
python gratitude.py
