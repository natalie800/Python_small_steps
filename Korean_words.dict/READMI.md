🇬🇧 English

A simple program that stores Korean words and their translations in a dictionary and displays them using a `for` loop.  
The user is asked whether they want to see the dictionary.

**What I practiced:**
- Creating a dictionary in Python
- Using a `for` loop with `.items()`
- User input with `input()`
- Conditional statement `if` with `.lower()`

🇷🇺 Русский

Простая программа, которая хранит корейские слова и их переводы в словаре и выводит их с помощью цикла for.
Пользователь отвечает на вопрос, хочет ли он увидеть словарь.

Тренируемые навыки:

· Создание словаря в Python
· Использование цикла for с методом .items()
· Ввод пользователя через input()
· Условный оператор if с преобразованием .lower()
